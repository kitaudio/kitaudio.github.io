Envelope

Envelope values are specified as 2D arrays in the form [ [t(0), a(0)], [t(1), a(1)], ... [t(i), a(i)] ], where t(i) specifies the time, in seconds, and a(i) specifies the amplitude of the envelope at the vertex i.
