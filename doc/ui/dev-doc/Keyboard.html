 true Boolean specifying whether the keyboard is editable by the mouse or touch interactions. A non-editable keyboard may be used as a visual diagram, for example.

(private) _initState()

State is represented as an array of active notes, each of which is an object { pitch, vel }, where pitch is MIDI pitch (0 - 127) and vel is MIDI velocity (0 - 127). A vel of 0 is reported once for each note-off event, and not reported on subsequent callback notifications.
